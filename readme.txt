ソース管理およびチーム作業の練習を目的に始まったプロジェクトです。
環境整備が進めば共同学習のサポートにも使えるかと思います。一石二鳥狙いです。

目的が学習そのものであるため基本コンセプトは下記のようになります。

○ 慣れない人が慣れないことをやる。
○ 既存のツールなどを使えば楽にできることも、可能な限りは自作を考える。
○ 使用言語やコードの内容についても同様で学習になるのであれば、寄り道、回り道を推奨する。

このドキュメントについても
「まずはテキストでコミットとか練習したらいいじゃない」
という回り道の思想が根底にあります。

できる人のための計画ではないというあたりが大事なところです。
失敗しても問題はないので積極的に挑戦しましょう。
